Why CivicNews?

Designed to be factual, verifiable, and completely private.

Daily Scan

Aggregates municipality signals automatically. Keeps you updated on the latest local government feeds without manual tracking.

Plain Language Summary & Rewrite

Converts dense, bureaucratic agendas and reports into clean, readable text. Full offline LLM support keeps your workspace private.

Source Tier Enforcement

Classify and segment feeds by reliability and official standing. Focus first on high-priority regulatory or legislative sources.

Bundled Ollama Sidecar

Features zero separate setup. Run your intelligence model directly on your CPU out-of-the-box, no external server needed. GPU acceleration is not yet bundled on Linux.

How It Works

A simple, repeatable cycle for objective reporting.

1

Scrape Sources

Link your local city hall, school board, or police department feeds. CivicNews pulls down their public documents securely.

2

Extract Signals

The detector engine scans the raw text and automatically creates "Leads" when it spots high-dollar contracts, personnel changes, or scheduled public hearings.

3

Draft & Verify

Your local Ollama assistant drafts an objective report. You then link every factual claim to the exact sentence in the raw municipal document to pass the Guardrails inspector.

4

Publish Statically

Use the built-in wizard to compile a beautiful flat HTML site (and, if you like, generate copy-ready social blurbs with the optional local-AI social-pack task). Drag and drop the folder to Netlify, Vercel, or GitHub Pages for free, fast hosting.

Technical Architecture

A secure, hybrid layout bridging desktop apps and browser extensions.

Sandboxed Local APIs

CivicNews runs entirely on the user's machine. It establishes an Axum loopback server strictly bound to 127.0.0.1:12053.

  • Tauri v2 Desktop Wrapper: React + TypeScript Glassmorphic UI connected via secure IPC.
  • SQLite WAL Database: Supports concurrent reads/writes for robust local data handling.
  • DNS Rebinding Guards: Strict host header validation prevents external network access.
  • Browser Pairing: Connect your Chrome extensions using secure PIN-pairing tokens to capture articles as you browse.
